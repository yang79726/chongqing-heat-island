# 重庆都市区热岛效应时空演变与驱动因素分析

## 📋 项目简介

本项目是**兰州大学地理信息科学专业**的课程实习作业，以重庆都市区为研究对象，基于2000-2025年Landsat长时序遥感影像，构建“数据制备-影像处理-分析制图”完整工作流，系统探究热岛效应的时空演变特征及驱动机制。

## 🎯 研究目标

- 揭示2000-2025年重庆都市区建成区扩张的时空演变规律
- 分析热岛效应的范围、强度变化及空间分布特征
- 量化地表要素（NDVI、NDISI、MNDWI等）与地表温度（LST）的关联
- 为山地城市热环境优化提供科学依据

## 🛠️ 技术栈

| 工具 | 用途 |
|------|------|
| **GEE** (Google Earth Engine) | 长时序Landsat影像处理、LST反演、地表指数计算 |
| **ArcMap** | 城区边界提取、矢量编辑、空间分析 |
| **R语言** | 统计分析、相关性分析、专题制图 |
| **ENVI** | 波段运算、图像分类辅助 |

## 📊 核心成果

### 1. 城区变迁特征
- 建成区呈 **“单中心集聚→双中心拓展→多中心扩散”** 态势
- 城区面积从 **32.0 km²** 扩张至 **89.2 km²**
- 空间格局：渝中半岛 → 渝中半岛+江北嘴 → 三足鼎立

### 2. 热岛效应演变
- 热岛范围扩张 **4.5倍**（45.2 km² → 208.6 km²）
- 热岛强度从 **8.2℃** 提升至 **15.1℃**
- 2015年后进入 **极强热岛** 等级

### 3. 驱动因素分析
- **NDVI** 是最敏感降温因子（r = **-0.77**）
- **NDISI** 是主导致热因子（r = **0.20**）
- 冷源：水体（MNDWI）、植被（NDVI）
- 热源：不透水面（NDISI）、建筑用地（NDBI）
